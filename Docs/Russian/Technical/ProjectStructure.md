# Структура проекта

```
HyPrism/
├── Program.cs                  # Точка входа: Console → запуск Electron
├── Bootstrapper.cs             # Настройка DI-контейнера
├── HyPrism.csproj              # Файл проекта с конвейером MSBuild
│
├── Frontend/                   # React SPA (Vite + TypeScript)
│   ├── src/
│   │   ├── components/         # Переиспользуемые React-компоненты
│   │   ├── pages/              # Компоненты-страницы маршрутов
│   │   ├── contexts/           # Провайдеры React Context
│   │   ├── lib/                # Утилиты
│   │   │   └── ipc.ts          # АВТОГЕНЕРИРУЕМЫЙ IPC-мост (не редактировать!)
│   │   ├── App.tsx             # Корневой компонент с маршрутизацией
│   │   ├── main.tsx            # Точка входа React
│   │   └── index.css           # Глобальные стили + Tailwind + токены темы
│   ├── public/
│   │   └── preload.js          # Preload-скрипт Electron
│   └── package.json            # Зависимости фронтенда
│
├── Services/                   # Слой .NET-сервисов
│   ├── Core/                   # Инфраструктурные сервисы
│   ├── Game/                   # Сервисы игровой логики
│   └── User/                   # Сервисы, связанные с пользователем
│
├── Models/                     # Модели данных (POCO)
├── Scripts/                    # Скрипты сборки и утилиты
├── Assets/                     # Ресурсы .NET-стороны (ЗАГЛАВНЫЕ БУКВЫ)
│   ├── Locales/                # JSON-файлы локализации (12 языков)
│   └── Images/
├── Packaging/                  # Пакетирование для ОС
├── Docs/                       # Документация
└── UI/                         # УСТАРЕЛО: Старый UI на Avalonia (НЕ использовать)
```

Важно: `Frontend/src/lib/ipc.ts` генерируется автоматически — никогда не редактируйте вручную. Директория `UI/` устарела.
