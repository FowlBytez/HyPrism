# Возможности

## Основные

- **Запуск игры** — Загрузка, установка, патчинг и запуск Hytale
- **Мультиинстанс** — Управление несколькими установками игры
- **Автообновление** — Автоматическое обновление лаунчера через GitHub Releases
- **Управление профилями** — Несколько профилей игроков с поддержкой аватаров

## Пользовательский интерфейс

- **Современный тёмный UI** — Безрамочное окно с дизайном в стиле glass-morphism
- **Анимации GSAP** — Плавные переходы между страницами и микровзаимодействия
- **Адаптивная компоновка** — Боковая панель навигации, панель информации, лента новостей, настройки, менеджер модов
- **Система тем** — CSS custom properties для полной настройки цветовой схемы

## Моддинг

- **Браузер модов** — Поиск и просмотр модов (интеграция с CurseForge)
- **Установленные моды** — Просмотр и управление установленными модификациями
- **Метаданные модов** — Версия, автор, описание, количество загрузок

## Социальные функции

- **Новости Hytale** — Лента последних новостей из официальных источников
- **Интеграция с Discord** — Rich Presence статус во время игры

## Интернационализация

- **12 языков** — en-US, ru-RU, de-DE, es-ES, fr-FR, ja-JP, ko-KR, pt-BR, tr-TR, uk-UA, zh-CN, be-BY
- **Переключение на лету** — Смена языка без перезапуска
- **Вложенные ключи** — Структурированная локализация с поддержкой подстановок ({0}, {1})

## Для разработчиков

- **Генерация IPC-кода** — Аннотации в C# автоматически генерируют типизированный TypeScript IPC-клиент
- **Конвейер MSBuild** — Автоматизированный npm install → генерация IPC → сборка Vite → копирование dist при `dotnet build`
- **Логирование Serilog** — Структурированное файловое логирование с выводом в консоль и перехватом логов Electron.NET
- **Пакетирование Flatpak** — Пакетирование для Linux с манифестом
