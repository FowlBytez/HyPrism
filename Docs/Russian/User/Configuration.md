# Настройка

HyPrism хранит свою конфигурацию в файле `config.json` внутри директории данных.

## Параметры

Доступ к настройкам осуществляется через страницу **Настройки** (иконка шестерёнки на боковой панели).

### Общие

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| Язык | Язык интерфейса | Язык системы или en-US |
| Закрывать после запуска | Закрывать лаунчер при старте игры | false |
| Запуск при старте системы | Автозапуск с ОС | false |
| Сворачивать в трей | Сворачивать в системный трей | false |

### Внешний вид

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| Цвет акцента | Акцентный цвет темы | Фиолетовый (#7C5CFC) |
| Анимации | Включить UI-анимации | true |
| Прозрачность | Эффекты glass-morphism | true |
| Режим фона | Стиль фона панели управления | default |

### Игра

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| Разрешение | Разрешение окна игры | 1920x1080 |
| Выделение RAM | Объём памяти для игры (МБ) | 4096 |
| Звук | Включить звук игры | true |

### Расширенные

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| Режим разработчика | Показывать инструменты разработчика | false |
| Подробное логирование | Расширенный вывод логов | false |
| Предварительные релизы | Получать предрелизные обновления | false |
| Ветка лаунчера | Канал release или pre-release | release |
| Директория данных | Пользовательский путь хранения данных | По умолчанию для платформы |

## Файл конфигурации

**Расположение:**
- Windows: `%APPDATA%/HyPrism/config.json`
- Linux: `~/.config/HyPrism/config.json`
- macOS: `~/Library/Application Support/HyPrism/config.json`

Файл конфигурации представляет собой JSON и может быть отредактирован вручную, однако рекомендуется использовать страницу настроек.

## Профили

HyPrism поддерживает несколько профилей игроков. Переключение между профилями осуществляется через селектор профилей на боковой панели. Каждый профиль хранит:
- Никнейм игрока
- UUID
- Аватар (опционально)
